## What this project is about

- **React basics and hooks**
  - `useState`, `useEffect`, `useContext`, `useReducer`, `useCallback`, `useMemo`, `useRef`
  - Example components live in `learn-space/hooksLearn/` and `learn-space/react-hooks-patterns/`

- **Routing with React Router**
  - Simple multi-page examples and nested routes
  - See `learn-space/reactRouter/`

- **State management with Zustand**
  - Lightweight global state patterns
  - See `learn-space/zustandLearn/`

- **Styling with Tailwind CSS**
  - Utility-first styling and small UI experiments
  - See `learn-space/tailwind/`

- **Build tooling with Webpack**
  - The app is bundled with **Webpack**
  - Custom config in `webpack.config.js`
  - Babel, PostCSS, and Tailwind are wired into the build

---

## Tech stack

- **React**
- **Webpack** (custom config)
- **Babel**
- **PostCSS + Tailwind CSS**
- **Jest** (for basic tests)
- **Zustand** (state management)
- **React Router** (routing)

---

## Goals

- Learn React concepts step by step
- Practice **clean, simple patterns** before using them in real projects
- Experiment safely without worrying about production code

---

## How to use this repo

- Browse the folders inside `learn-space/` to see different topics
- Open the example files and read the code
- Run the dev server and play with the UI as you change things

> This repo is mainly for **learning**, not for production.
